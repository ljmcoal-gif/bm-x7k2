# 펀딩 구조 v2 — 재무팀 의견 반영 (BA 보조금-우선 / OT 링펜스 유지)

> 이 문서는 `핵심결론_종합노트.md`의 §8~12(파이낸싱)를 **재무팀 의견에 따라 전면 개정**한 v2 설계다.
> 핵심 전환: **정부 앵커를 OT → BA로 이동**, BA는 **보조금(grant) 우선**, OT의 DOE 링펜스는 **유지**.
> ※ ★ 표시는 미검증 입력 — 데이터룸/법률/DOE 확인 필요. 확정 전까지는 민감도로 범위 제시.

---

## 0. 재무팀 의견 (입력)

1. **Feasibility (BA vs OT)**: OT는 ATVM 수혜를 이미 받고 있어, DOE/DOW 등 미 정부 신규자금은 **BA측 feasibility가 더 높다**고 판단.
2. **BA 조달수단**: BA 개조(retrofit) 자금은 **정부 보조금을 최대한 선호**.
3. **Loan 제약**: BA는 이미 차입금이 많음(**약 $5bn**). **SK이노베이션 보증·확약(covenant) 관리** 필요.
4. **Equity 제약**: BA 밸류에이션·지분희석·CoC 고려 필요. **DOE의 Equity 투자 가능 여부 불확실.**
5. **프로세스**: DOE는 **어느 부서와 미팅하기 전 재무와 사전 Comm. 필수.**

---

## 1. 한 줄 결론

> **정부 앵커를 OT에서 BA로 옮기되, BA는 '대출'이 아니라 '보조금 우선'으로 짜고, Equity는 잔여·최후수단으로만 둔다. OT의 DOE 링펜스(§9)는 건드리지 않는다.**

근거: 보조금은 **DSCR·CoC·지분희석이 모두 없는 유일한 자금**이다. EBIT=0인 박리다매 자산(§12)에는 상업대출이 원천적으로 안 붙으므로, "갚을 필요 없는 돈(grant)"을 BA 개조비에 최대한 끌어오는 것이 feasibility뿐 아니라 **구조적으로도 최적**이다.

## 2. v1 → v2 무엇이 바뀌나

| 항목 | v1 (기존 모델) | v2 (재무팀 반영) |
|---|---|---|
| **지배구조** | HoldCo → [BA]·[OT] **형제(추정)** | **SK이노 → BA → OT (BA가 OT 100% 보유, 모자구조)** |
| 정부자금 타깃 | **OT** (DOE $4bn ATVM) | **BA** (OT는 ATVM 소진 → BA feasibility ↑) |
| BA 성격 | 상대적으로 "clean" → 외부 Equity ≤19.9% 수용처 | **BA 단독 ~$5bn 차입 + SKI 보증** → clean 아님 |
| BA 조달 1순위 | (명시 안 됨) | **보조금(grant)** |
| 신규 Loan | — | **회피** (BA 레버리지 한도·SKI 보증 관리) |
| Equity | BA로 ≤19.9% | **잔여·최후수단**, DOE Equity = 불가 가정 |
| OT 구조 | DOE 담보 + 5층 링펜스 | **그대로 유지** (모자 격리로 재해석) |
| 세금 | 법인별 분리 과세 | **연결납세** — OT 적자가 BA 소득 상계 |

### [핵심 인식] 비대칭의 붕괴
v1의 **"BA=깨끗 / OT=묶임"** 전제가 무너진다. v2에서는 **양쪽 다 묶인 공장**이다 — OT는 DOE 담보로, BA는 **단독** $5bn 부채 + SKI 보증으로. 따라서 **격리·확약 관리 논리를 BA에도 적용**해야 한다(기존엔 OT 전용이었음).

## 2-0. 조달 메커니즘 — BA 일괄조달 → OT 증자

**BA가 $1.4bn 전액을 일괄 조달**한다(외부: 보조금 + 비소구부채 + Equity). 그중 **$0.4bn은 BA 자체 개조에 보유**, **$1.0bn은 OT에 증자(equity-down)** 한다. OT의 기존 DOE $4.0bn ATVM은 별도 선순위로 유지.

- **왜 OT에 '부채'가 아니라 '증자'인가**: OT는 DOE 선순위·CoC·permitted-debt 제약으로 신규 차입이 막힘 → **모회사 증자가 자회사 자금주입의 가장 깨끗한 경로**(신규 OT 부채·DSCR·intercreditor 불요, 100% 자회사라 CoC 무관). 기존 메자닌(§11)의 DSCR 벽 문제도 소멸.
- **유의점 ★**: $1.4bn 조달 부담이 **전부 BA(이미 단독 ~$5bn + SKI 보증)에 집중** → 신규 부채여력 빠듯 → 그랜트 + (한도내)비소구부채 + Equity(<20%)로 맞춰야 함. 또 OT 몫이 메자닌(7%)→BA 에쿼티(Ke 13%)로 바뀌면 자본비용↑·회수는 OT FCFE 의존 → §12(EBIT=0) 문제로 회귀. **구조는 깨끗, 부담은 BA/지분으로 이동.**

---

## 2-1. 지배구조 — BA → OT 완전자회사 (모자구조)

**확정 사실**: OT(SKOT)는 **BA(SKBA)의 100% 완전자회사**다. 두 공장은 형제가 아니라 **모자(母子)관계**:
`SK이노베이션 → BA (중간지주·조달 주체·보조금 수령) → OT (완전자회사 100% · DOE 격리)`.

이로 인해 달라지는 4가지:

1. **Equity 게이트가 'BA 하나'로 단일화** — BA가 OT를 100% 소유하므로 **BA 지분 외부 매각 = OT(DOE 담보)의 upstream 지배권 변동** → DOE CoC 트리거. 즉 기존의 "BA 지분 20%↑ → CoC"는 **바로 이 모자구조 때문**. 외부 자본 진입점이 BA로 모여 단순하지만 **<20% 캡을 더 강하게** 지켜야 함.
2. **링펜스 = '모자 격리'로 재해석 (여전히 필요)** — OT의 DOE $4bn은 **BA에 non-recourse** → OT가 2033년 후 무너져도 BA 손실은 **BA가 보유한 OT 지분가치까지로 한정**. 거꾸로 BA 단독 $5bn(+SKI 보증)은 **OT가 보증·담보 제공 못 하게** 차단(DOE separateness가 금지) → **양방향 격벽**.
3. **연결납세 업사이드 (모자구조 효과)** — 완전자회사(美 80%+)면 **연결납세** 대상 → **OT의 적자가 BA 과세소득을 상계** → 그룹 현금세금↓. 단 `핵심결론노트 §6`대로 **BA 과세소득 자체가 작아 절대 규모는 제한적**(세금이 이미 거의 0). 방향은 +, 크기는 작음 → 모델에서 정량 확인 ★.
4. **자금 흐름이 '정상 모회사 출자'로** — 모자구조에선 BA→OT 다운스트림이 정상 출자(우회 아님). 보조금은 BA 개조용이면 BA에 머물고, OT 보강이 필요하면 다운스트림 출자(추가 equity는 OT 강화일 뿐 CoC 트리거 아님).

## 3. BA 펀딩 워터폴 (조달 우선순위)

BA 개조비를 채우는 순서를 명확한 위계로 둔다.

### ① 보조금 (1순위 — 최대화) ★
- **후보 비히클**: DOE MESC(제조·공급망) grant, 48C 첨단에너지 투자세액공제(현금화 가능), 주(州) 인센티브.
- **장점**: 무상환·무DSCR·무희석·CoC 무관. EBIT=0 자산에 유일하게 "깨끗한" 자금.
- **확인 필요**: 적격성·한도(법률/DOE), BA 개조가 해당 프로그램 정의에 부합하는지.

### ② 비소구(non-recourse) 자산기반 부채 (2순위 — 제한적)
- **담보**: BA 설비·재고 + **45X AMPC 수취권 유동화**(IRA상 양도 가능).
- **레드라인**: **SKI 보증 스택에 얹지 않는 구조**여야 함(이미 $5bn 보증 부담). 기존 차입계약의 leverage/financial covenant **여유 범위 안에서만**, cross-default 회피.

### ③ Equity (3순위 — 잔여·최후)
- **사이징**: **20% CoC 캡 아래**로 엄격히. 밸류에이션 플로어·anti-dilution 부착.
- **DOE Equity = 불가 가정 ★**: DOE LPO/ATVM은 법령상 대출·보증이지 지분이 아님 → 재무팀의 "불확실" 직관과 일치. **기대치를 빼고 설계**한다.

### [원칙] 워터폴 적용
조달은 ① → ② → ③ 순서로 채운다. 상위 단계로 충당 가능한 금액은 하위 단계로 내리지 않는다(희석·부채부담 최소화).

---

## 4. OT는 그대로 둔다

OT의 **DOE $4bn ATVM 선순위 담보 + 5층 링펜스(§9) + CoC**는 유지. OT 추가 소요는 기존 계획대로 **AMPC 현금화 + 메자닌(보조금 창 2029~32 내 상환)**으로 충당(§11). 정부 앵커를 BA로 옮기는 것이지, **OT 구조를 푸는 것이 아니다.**

## 5. DOE 접근 시퀀스 (재무팀 사전 Comm 반영)

```
1. 내부 재무 정렬 (선행, 필수)
   - 1페이지 얼라인먼트 메모: ask(grant vs loan) · 구조 · 레드라인
     (SKI 보증 확대 금지 / BA covenant breach 금지 / CoC 20% 유지)
2. ask에 맞는 DOE 카운터파트 매칭
   - 보조금 → MESC(제조·공급망) 라인
   - 대출   → LPO(ATVM/Title 17)   ※ OT가 이미 여기
   - Equity → 해당 부서 없음 (DOE는 지분투자 안 함)
3. DOW(국방) 앵글 → 방산·딥테크 슬리브로 별도 트랙
```

### [원칙] 부서 미팅 전 ask를 먼저 합의
"무엇을 요청할지(grant/loan/규모)"를 **재무와 먼저 확정**한 뒤 매칭되는 부서로 간다. 잘못된 부서에 잘못된 ask로 가는 것을 막는다.

## 7. 확정 사실 & 남은 ★ 미검증 항목

### ✅ 확정 (재무팀)
- **지배구조**: OT는 BA의 **100% 완전자회사** (모자구조).
- **$5bn 차입**: **BA 단독** (OT의 DOE $4bn 미포함).
- **연결납세**: **적용** — OT 적자가 BA 과세소득 상계 (규모는 §6대로 제한적).

### ★ 남은 미검증 (모델 정량화 전제)
1. **BA 단독 $5bn의 상세** — 건설/텀론 여부, SKI 보증 범위, financial covenant(레버리지·DSCR 한도).
2. **BA 개조 CapEx** — DC블록/LFP 전환 비용 금액(보조금 사이징의 분모).
3. **보조금 적격성·규모** — MESC/48C 실제 수혜 가능액.
4. **CoC 재조정** — 기존 BA 외부 Equity ≤19.9% 슬리브가 보조금 중심 전환 시 축소/소멸 → 풋볼필드·NAV 영향.
5. **연결납세 정량 효과** — OT 적자×BA 소득 상계의 EV/FCFE 기여(§6: BA 과세소득이 작아 소폭 예상).

> 위가 확정되기 전에는 **민감도 범위**로 제시한다(예: 보조금 $0.5~1.5bn, BA 단독 부채 $5bn 고정, Equity 0~19.9%).
